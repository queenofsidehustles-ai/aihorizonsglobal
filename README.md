# AI Horizons Global Consultants — Website

The official website for **AI Horizons Global Consultants**, an AI and operations
transformation practice led by founder **Monica Lewis**. We help mid-size businesses
and government agencies find where they lose time and money — then deploy AI that fits
how their people already work.

🌐 **Live site:** [aihorizonsglobal.com](https://aihorizonsglobal.com)

---

## What's here

A fast, static marketing site — plain HTML and CSS, no build step required.

| Page | Purpose |
|------|---------|
| `index.html` | Home — positioning, approach, and contact form |
| `sprint.html` | The AI Readiness Sprint — our signature 2-week engagement |
| `case-studies.html` | Portfolio of real transformations delivered |
| `founder.html` | Monica Lewis — background and credentials |
| `services.html` | Full service catalog (business & government tracks) |
| `innovation.html` | Innovation Lab — AI prototypes (unlinked / archive) |
| `images/` | Logos, hero art, and illustrations |
| `sitemap.xml`, `robots.txt` | Search-engine indexing |

## Tech

- Static HTML + CSS (fonts: Playfair Display + Inter)
- Brand palette: navy `#0a1435` and gold `#d4a740`
- Hosted on **Netlify**, auto-deployed from the `main` branch on GitHub
- Contact form powered by **Netlify Forms**

## How to update the site

1. Edit any `.html` file (on GitHub.com or locally).
2. Commit the change to the `main` branch.
3. Netlify automatically publishes it within a minute or two — no manual upload.
